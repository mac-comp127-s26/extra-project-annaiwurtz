# COMP 127 Extra Project: Flappy Bird

This is my extra project for COMP 127, where I implemented a basic version of the Flappy Bird game. This game involves a bird that flaps upward when the user clicks, trying to travel through the gaps in the pipes. After passing each pipe, the user's score is incremented by one. The game ends when the bird either collides with a pipe or hits the ground.

If I were to improve this project, I would figure out how to add graphics for a bird or another animal, since it is currently only a circle. I would also add a “Play Again” button, because I decided to leave it out after facing a lot of errors while implementing it.
